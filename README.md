# Pokedex

## English

### Overview
This project is a simple web application that uses the PokeAPI to display a catalog of the first generation of Pokémon. It allows users to browse Pokémon cards, search by name or number, filter by type, and open a detailed modal with additional information.

### Features
- Browse the first 151 Pokémon
- Search by name or number
- Filter by Pokémon type
- View detailed information in a modal
- Display basic stats, abilities, habitat, description, and moves

### Technologies
- HTML
- CSS
- Vanilla JavaScript
- PokeAPI

### How to run locally
1. Clone the repository.
2. Open the project folder.
3. Open the file index.html in your browser.

If you prefer a local server, you can run:

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### Project structure
- index.html — main page structure and JavaScript logic
- styles.css — visual design and responsive layout

## Português

### Visão geral
Este projeto é uma aplicação web simples que utiliza a PokeAPI para exibir um catálogo dos Pokémon da primeira geração. Ela permite navegar pelos cards dos Pokémon, buscar por nome ou número, filtrar por tipo e abrir um modal com informações detalhadas.

### Funcionalidades
- Navegar pelos 151 primeiros Pokémon
- Buscar por nome ou número
- Filtrar por tipo
- Ver informações detalhadas em um modal
- Exibir estatísticas básicas, habilidades, habitat, descrição e movimentos

### Tecnologias
- HTML
- CSS
- JavaScript puro
- PokeAPI

### Como executar localmente
1. Clone o repositório.
2. Acesse a pasta do projeto.
3. Abra o arquivo index.html no navegador.

Se preferir utilizar um servidor local, você pode executar:

